# ⏱️ Stopwatch Application

## 📌 Project Description
The Stopwatch Application is a simple and user-friendly tool designed to measure and track time accurately. It allows users to start, pause, reset, and monitor elapsed time with precision.

## 🚀 Features
- Start the stopwatch
- Pause/Stop the timer
- Reset the timer
- Accurate time tracking
- Simple and clean user interface

## 🛠️ Technologies Used
- HTML
- CSS
- JavaScript

## ▶️ How to Run the Project
1. Download or clone the repository.
2. Open the project folder.
3. Double-click the `index.html` file.
4. The stopwatch will open in your browser.
